解法1:
[... X X i] X X
dp[i]為當前竊取的最大利益
每次可以選擇偷或不偷
dp[i] = max(dp[i - 1], nums[i] + dp[i - 2])

解法2:
可以用1個變數紀錄當下最大值
nums = [2, 7, 9, 3, 1]

(L, M) = (0, 2)
(L, M) = (2, 7)
[2, 7, 11, 0, 0] => (L, M) = (7, 11)
[2, 7, 11, 11, 0] => (L, M) = (11, 11)
[2, 7, 11, 11, 12] => (L, M) = (11, 12)

dp   =  [2, 7, 11, 11, 12]
nums =  [2, 7, 9,  3,  1]
i = 0   (M)
i = 1      (M)
i = 2    L  X  i
              (M)
i = 3       L  X   i
                  (M)
i = 4           L  X   i
                      (M)

M = dp[i]
